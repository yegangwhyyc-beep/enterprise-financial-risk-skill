# 安永财务风险评估指标规范
## 一、核心指标预警阈值
| 指标类型 | 指标名称 | 安全阈值 | 预警阈值 | 高危阈值 |
| --- | --- | --- | --- | --- |
| 偿债能力 | 资产负债率 | <60% | 60%-75% | >75% |
| | 流动比率 | >1.5x | 1.2x-1.5x | <1.2x |
| | 速动比率 | >1x | 0.8x-1x | <0.8x |
| | 利息保障倍数 | >5x | 3x-5x | <3x |
| | 有息负债率 | <30% | 30%-50% | >50% |
| 盈利能力 | 毛利率 | >20% | 15%-20% | <15% |
| | 净利率 | >5% | 3%-5% | <3% |
| | 经营现金流/净利润 | >1.2x | 0.8x-1.2x | <0.8x |
| 运营效率 | 应收账款周转率 | >8次/年 | 5-8次/年 | <5次/年 |
| | 存货周转率 | >6次/年 | 3-6次/年 | <3次/年 |
| | 产能利用率 | >70% | 50%-70% | <50% |

| | 现金流覆盖刚性支出时长 | >6个月 | 3-6个月 | <3个月 |

## 二、风险分级标准
采用"风险概率×影响程度"矩阵法划分风险等级：
| 风险等级 | 标识 | 定义 | 响应要求 |
| --- | --- | --- | --- |
| 极高风险 | 红色 | 发生概率>30%，且影响程度为重大（可能导致集团层面违约、现金流断裂、重大损失） | 立即启动专项处置，制定应对方案，每周跟踪整改进度 |
| 高风险 | 橙色 | 发生概率10%-30%，或影响程度为较大（可能导致板块层面亏损、融资受阻） | 重点监控，制定整改计划，每半月跟踪整改进度 |
| 中风险 | 黄色 | 发生概率5%-10%，或影响程度为一般（可能导致局部业务波动） | 持续跟踪，每季度评估风险变化 |
| 低风险 | 蓝色 | 发生概率<5%，影响程度较小 | 常规监控，年度风险排查 |

## 三、模型参数规范
1. **Altman Z-score优化版（制造业）**：Z = 1.2X1 + 1.4X2 + 3.3X3 + 0.6X4 + 0.999X5
   - X1=营运资本/总资产，X2=留存收益/总资产，X3=息税前利润/总资产，X4=权益市值/总负债，X5=销售收入/总资产
   - Z>2.99：安全区，Z=1.81-2.99：灰色区，Z<1.81：高危区
2. **Ohlson O-score**：O<-1.2：低风险，O=-1.2-0.5：中风险，O>0.5：高风险（财报异常概率>50%）
3. **Merton模型PD阈值**：PD<0.5%：投资级低风险，PD=0.5%-3%：投机级中风险，PD>3%：高风险
