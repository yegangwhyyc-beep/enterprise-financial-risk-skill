# 企业财务风险预测技能（安永标准）
本技能基于安永20年大型企业财务风控服务经验构建，可提供专业级企业财务风险预测服务，通过"识别-评估-预警-优化"全流程，提前发现并防范现金流断裂、偿债违约、盈利能力恶化等财务危机，输出符合监管要求与管理层决策需求的专业风控报告。

## 全流程自动化工作流
### 🚀 一键执行方式
```bash
# 1. 准备输入数据文件（JSON格式，参考input.example.json）
# 2. 执行主脚本，自动完成全流程分析
cd scripts && python3 main.py [input.json]
```
执行完成后将自动在`output_企业名_日期`目录下生成6份PDF报告：
1. 《全维度财务风险清单》
2. 《财务风险量化评估报告》
3. 《压力测试与情景模拟报告》
4. 《财务风险应对方案》
5. 《财务风险应对方案》（可选重复版本）
6. 《集团财务风险预测与管控报告》

### 阶段1：数据导入与预处理（自动完成）
自动读取输入财务数据，完成数据标准化校验、行业对标数据匹配
**输出物**：财务数据标准化报告、行业对标分析结果

### 阶段2：核心风险识别（自动完成）
采用"财务数据+业务场景+内控流程"三维识别法，覆盖显性+隐性风险：
1. **显性风险**：偿债风险、盈利风险、运营风险、市场风险（汇率/利率/价格波动）
2. **隐性风险**：并购整合风险、内控缺陷风险、合规风险、战略落地风险
**输出物**：《全维度财务风险清单》，标注风险等级（红/橙/黄/蓝）、影响范围、潜在冲击程度

### 阶段3：风险量化评估与建模（自动完成）
使用四大专业财务风险模型开展量化测算：
1. **优化版Altman Z-score模型**：测算破产风险区间（安全区/灰色区/高危区）
2. **Ohlson O-score模型**：识别财报异常、会计舞弊风险
3. **Merton模型**：测算违约概率（PD）、违约损失率（LGD）
4. **三级现金流预测模型**：搭建"集团-板块-子公司"联动现金流模型，滚动预测未来12-24个月现金流缺口与峰值压力
5. 开展多维度敏感性分析，测算不同场景下的风险冲击程度
**输出物**：《财务风险量化评估报告》、风险分级清单

### 阶段4：压力测试与情景模拟（自动完成）
设计三类情景测试企业抗风险能力：
1. 基准情景：正常经营状态下的风险变化
2. 不利情景：行业下行、营收下滑15%-20%、成本上升等中度压力情景
3. 极端情景：黑天鹅事件、核心业务崩盘等极端压力情景
**输出物**：《压力测试与情景模拟报告》，明确风险临界点与安全缓冲建议

### 阶段5：应对方案与最终交付（自动完成）
1. 针对高风险/极高风险项自动生成专项应对方案，明确责任部门、执行时限、考核标准
2. 整合所有成果自动生成6份PDF格式专业报告，包含封面、目录、图表、签名页，符合安永对外交付规范，适配管理层决策需求与董事会汇报要求
**输出物**：6份PDF报告 + 可编辑版本原始报告文件
